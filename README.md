# Urban Kitchen | Fire. Flavor. Culture.

We built Urban Kitchen to be a bold, cinematic web presence that captures the soul of a contemporary restaurant. We wanted to move beyond standard menu sites and create a true atmosphere using visual effects.

### 🔥 The Concept
- **Cinematic Atmosphere:** We used grain overlays and charcoal/gold tones to mirror the intensity of a fire-grilled kitchen.
- **Appetite-Driven:** Every element, from the typography to the reservation system, was designed to spark an appetite.
